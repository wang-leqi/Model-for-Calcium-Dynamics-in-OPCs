 # Modeling OPC calcium 
 ## Subfolders: 
 - **data**: Prolonged ATP/Glutamate stimulation Calcium imaging data from [Paez Laboratory](https://www.buffalo.edu/myelin-glia-exploration/laboratories/Paez-lab.html). Data generated by [Veronica T. Cheli](https://scholar.google.com/citations?user=vfCGQrcAAAAJ&hl=en).
 - **python**: This folder contains the Python code for the OPC model and generates time series simulations.
 - **xpp** xpp code to generate bifurcation diagram.
